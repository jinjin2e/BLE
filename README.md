# BLE
BLE, Bluetooth Low Energy 블루투스 4.0을 사용하는 저전력 디바이스를 말한다.  
저전력 디바이스인 만큼 고속통신에 단점이 있음. 대신 Coin Cell 하나만으로도 1년정도 동작 할 정도로 배터리 소모량이 적다.
